# v0.0.2-beta — In progress

## Overview

Ongoing development and refinements for the Lux compiler and its ecosystem. This release focuses on developer experience improvements and stability.

## What's New

### Tooling

#### VS Code Extension
- **Doc-Comment Auto-Prefix**: Added automatic ` * ` prefixing for doc-comments (`/** ... */`) when pressing Enter. This improves documentation ergonomics by maintaining the standard block-comment style automatically.
